### Installation

<div id="installation"></div>

1. Clone the repo

   ```sh
   git clone https://github.com/itshosein/itshosein-server.git
   ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. For development

   ```js
   npm run dev
   ```

   For production

   ```js
   npm run build
   npm start
   ```
